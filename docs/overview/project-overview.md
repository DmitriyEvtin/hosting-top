# Обзор проекта

[← Назад к документации](../README.md)

## Описание

Система каталога металлопроката с автоматическим парсингом данных с сайта bvb-alyans.ru, админ-панелью для управления товарами и интеграцией с AWS S3 для хранения изображений.

## Цели проекта

- **Автоматизация**: Автоматический сбор данных о товарах металлопроката
- **Централизация**: Единая система управления каталогом товаров
- **Масштабируемость**: Возможность добавления новых источников данных
- **Производительность**: Быстрый поиск и фильтрация товаров
- **Удобство**: Современный пользовательский интерфейс

## Целевая аудитория

- **Покупатели металлопроката** - поиск и сравнение товаров
- **Администраторы** - управление каталогом и мониторинг парсинга
- **Разработчики** - поддержка и развитие системы

## Ключевые метрики

- **Покрытие каталога**: 100% товаров с bvb-alyans.ru
- **Время парсинга**: < 2 часов для полного обновления
- **Доступность**: 99.9% uptime
- **Производительность**: < 2 сек время загрузки страниц

## Бизнес-ценность

- **Экономия времени**: Автоматизация ручного ввода данных
- **Актуальность**: Регулярное обновление информации о товарах
- **Конкурентоспособность**: Быстрый доступ к актуальным ценам
- **Масштабируемость**: Возможность добавления новых поставщиков
