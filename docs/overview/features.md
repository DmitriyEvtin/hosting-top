# Основные возможности

[← Назад к документации](../README.md)

## Парсинг данных

### Автоматический сбор данных

- **Парсинг каталога** - Автоматический сбор товаров с bvb-alyans.ru
- **Обновление данных** - Регулярное обновление информации о товарах
- **Мониторинг изменений** - Отслеживание изменений на сайте-источнике
- **Обработка ошибок** - Автоматическое восстановление после сбоев

### Обработка изображений

- **Скачивание изображений** - Автоматическое скачивание изображений товаров
- **Оптимизация** - Сжатие и оптимизация изображений
- **Миниатюры** - Генерация миниатюр разных размеров
- **CDN интеграция** - Загрузка в AWS S3 с CloudFront

## Пользовательский интерфейс

### Каталог товаров

- **Поиск товаров** - Быстрый поиск по названию и характеристикам
- **Фильтрация** - Фильтрация по категориям, ценам, характеристикам
- **Сортировка** - Сортировка по различным параметрам
- **Пагинация** - Удобная навигация по страницам

### Детальная информация

- **Карточки товаров** - Подробная информация о каждом товаре
- **Галерея изображений** - Просмотр изображений товара
- **Характеристики** - Технические характеристики и параметры
- **Связанные товары** - Рекомендации похожих товаров

## Админ-панель

### Управление товарами

- **CRUD операции** - Создание, редактирование, удаление товаров
- **Массовые операции** - Массовое редактирование товаров
- **Импорт/экспорт** - Загрузка и выгрузка данных
- **Валидация данных** - Проверка корректности данных

### Управление категориями

- **Иерархия категорий** - Многоуровневая структура категорий
- **Управление SEO** - Настройка мета-данных для SEO
- **Сортировка** - Настройка порядка отображения категорий

### Мониторинг парсинга

- **Статус парсинга** - Отслеживание процесса парсинга в реальном времени
- **Логи парсинга** - Детальные логи процесса парсинга
- **Статистика** - Статистика успешности парсинга
- **Уведомления** - Уведомления о завершении или ошибках парсинга

## Система аутентификации

### Безопасность

- **JWT токены** - Безопасная аутентификация
- **Роли пользователей** - Разграничение доступа по ролям
- **Сессии** - Управление пользовательскими сессиями
- **Аудит** - Логирование действий пользователей

### Управление пользователями

- **Регистрация** - Самостоятельная регистрация пользователей
- **Профили** - Управление профилями пользователей
- **Восстановление паролей** - Безопасное восстановление доступа

## Производительность

### Оптимизация

- **Кэширование** - Кэширование часто запрашиваемых данных
- **Lazy loading** - Ленивая загрузка изображений и компонентов
- **Code splitting** - Разделение кода для быстрой загрузки
- **CDN** - Использование CDN для статических ресурсов

### Мониторинг

- **Метрики производительности** - Отслеживание времени загрузки
- **Мониторинг ошибок** - Автоматическое отслеживание ошибок
- **Аналитика** - Статистика использования системы
