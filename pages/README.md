# Pages Directory

Эта папка создана для совместимости с Next.js, даже при использовании App Router.

## Зачем нужна эта папка?

- **Next.js требование**: Next.js ожидает наличие папки `pages/` в корне проекта
- **Совместимость**: Обеспечивает совместимость с различными конфигурациями Next.js
- **Предотвращение ошибок**: Избегает ошибок сборки и предупреждений

## Структура проекта

```
├── pages/                    # Пустая папка (требование Next.js)
│   └── README.md             # Этот файл
├── src/                      # Основной код приложения
│   ├── app/                  # Next.js App Router
│   ├── pages/                # FSD Pages слой
│   └── shared/               # FSD Shared слой
```

## Важно

- **Не добавляйте файлы** в эту папку - она должна оставаться пустой
- **Используйте** `src/pages/` для FSD Pages слоя
- **App Router** использует `src/app/` для маршрутизации

Эта папка существует только для предотвращения ошибок Next.js и не содержит функционального кода.
